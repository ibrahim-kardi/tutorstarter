== Description ==
Tutor Starter is a super fast, easily customizable, multi-purpose theme brought to you by Themeum.

== Installation ==
= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "TutorStarter"
3. Install and activate
